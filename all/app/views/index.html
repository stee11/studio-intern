Congratulations, you're running a node app.

Goal: Create an application that polls the server for new data on an interval and updates the frontend page with that data

  • Create a new route ("/api/number") on the backend
  • Create a new controller that will interact with the route you just created
  • The response to the front end should be JSON (use this.jsonCollectionOutput())
  • Have the response generate a random number
  • Update app.js to perform AJAX requests poll the server on an interval
  • Have the response update the "target" element on this page using jQuery

Notes:

  • Route creation is done in /all/app/config/routes.js
  • Controllers are found in /all/app/controllers
  • app.js is found in /all/app/public/static/js
